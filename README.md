# Pharmacy Management System

Đồ án quản lý nhà thuốc viết bằng Python và SQL Server.

## Chức năng

- Quản lý thuốc
- Quản lý hóa đơn
- Quản lý khách hàng
- Quản lý nhập hàng
- Thống kê doanh thu

## Công nghệ

- Python
- SQL Server
- Tkinter
- pyodbc

## Cài đặt

pip install pyodbc

## Chạy chương trình

python Pharmacy.py
